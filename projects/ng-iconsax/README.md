# Ngiconsax

## Configuration

Add in `assets` on `angular.json`

```
{
    "glob": "**/*",
    "input": "./node_modules/ngiconsax/assets",
    "output": "/assets/"
}
```

Add import `import {NgIconSaxModule} from "ngiconsax"
` in  `your module`

## Use Example

```
<iconx name="3d-cube-scan" width="500" height="500" color="9f85ff" iconStyle="linear"></iconx>

```
